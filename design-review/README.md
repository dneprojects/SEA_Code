# SEA — Design-Mockup (Review)

Statisches, self-contained Mockup des neuen UI mit Dummy-Daten. **Noch nichts an der echten App geändert.**

## Öffnen
`design-review/app.html` einfach im Browser öffnen (Doppelklick). Keine Server/Abhängigkeiten nötig.

## Was du testen kannst
- **Oben die Review-Leiste:** Header-Variante **1–7** umschalten (live).
- **Menü links:** alle 8 Seiten durchklicken (Dashboard, Verlauf, Strategien, Einsparung, Geräte, Grundeinstellungen, Erzeuger/Verbraucher, Temperaturabsenkung).
- **Desktop:** festes 1024-px-Layout; Fenster schmaler ⇒ horizontaler Scroll (kein Umbruch).
- **Mobil:** Fenster auf < 760 px ziehen ⇒ Menü klappt zum Hamburger ein, Inhalte einspaltig/hochkant.
- **ⓘ bei den Strategien:** öffnet die Beschreibung im Overlay.

## Header-Varianten (alle mit dem roten Verlauf eingefärbt)
1. **Solar-Landschaft** — Sonne (Linienkunst) + Hügel + PV-Panel.
2. **Smart-Home Flow** — Sonne → Haus → Speicher/Netz mit animierten Energie-Flusslinien.
3. **Sunburst** — radiale Sonnenstrahlen, minimal/modern.
4. **Energiewellen** — übereinanderliegende Leistungs-Wellen.
5. **Tech-Hexgrid** — Smart-Grid-Wabenmuster, nach rechts ausblendend.
6. **Duotone Skyline** — Stadtsilhouette + Sonne, zweifarbig.
7. **Windräder & Sonne** — Windkraft + Sonne über Hügel.

> Sag mir, **welche Header-Nummer** dir gefällt (gern auch Kombi-Wünsche), dann baue ich genau diese in die echte App ein und setze das Layout produktiv um. Die Mockup-Dateien werden danach wieder entfernt.

## Designprinzipien im Mockup
- Einheitliche Button-Höhe (34 px; Icon-Buttons quadratisch), konsistente Karten/Radien/Schatten.
- Einrichtung über **Zeilen-/Spalten-Grid ohne Tabellenlinien** (Label · Entität · Live-Wert · Aktion sauber ausgerichtet).
- Durchgängiges Inline-SVG-Icon-Set (keine externen Abhängigkeiten).
